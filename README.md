# ☕ Brew & Bloom Café Website

A modern and responsive café website built using **HTML, CSS, and JavaScript**.
This project is created as part of the **Future Interns Full Stack Web Development Task** to design a professional website for a local business and host it live on GitHub.

---

## 🌐 Live Website

🔗 https://rakshita-patil25.github.io/Future_FS_03/

---

## 📌 Project Overview

**Brew & Bloom Café** is a stylish and aesthetic café website that showcases:

* Home section with hero banner
* About café section
* Coffee menu
* Vegan menu
* Cold drinks menu
* Reserve table feature
* Contact section
* Google Map location
* Responsive and modern UI

This website is designed to look like a real café business website that can be presented to a café owner.
